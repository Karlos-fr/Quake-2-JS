# Quake II Demo Assets

This directory contains assets extracted from the official Quake II 3.14 demo installer:

```text
q2-314-demo-x86.exe
MD5: 4D1CD4618E80A38DB59304132EA0856C
```

The extracted runtime data used by Quake2JS is:

```text
baseq2/pak0.pak
baseq2/players/
```

These files are included so the GitHub Pages deployment can start with demo/shareware game data. Retail Quake II assets are not included.
